Tres perfiles: Frontend, Backend y Bases de datos

Frontend: html, css (Stylus) y js (jQuery). Parte con la que el usuario interactua. Css también sirve para estilar Java y otros lenguajes. Único lenguajes de programación para Frontend.

Backend: es la parte de programación. PHP, .NET, ASP 3.0, Java, Ruby, Python (Django)

Bases de datos: MySQL (relacional), Postgres, MongoDB (No relacional)
